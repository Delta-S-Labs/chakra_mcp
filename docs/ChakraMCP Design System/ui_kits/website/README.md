# ChakraMCP — Website UI kit

Marketing site recreation: Portfolio landing, Concept page, Developer page stub.

**Open** `index.html` — top nav switches between the three pages.

- `Shell.jsx` — sticky header, footer, RelayDiagram
- `PortfolioPage.jsx` — hero, audience lanes, highlights, flow, fact sheets, closing panel
- `ConceptPage.jsx` — at-a-glance + chaptered stages (objects, flow, consent)
- `site.css` — all site chrome; imports `colors_and_type.css` for tokens

Source of truth: `app/src/` in the project root (imported reference codebase).
